All notable changes to this extension will be documented in this file.

## [1.0.0] - 2026-04-01

### Added

- Initial release of Spec Critique extension
- Command: `/speckit.critique.run` - Dual-lens spec and plan review
- Product lens: user impact, scope, alternatives, competitive analysis
- Engineering lens: architecture risks, security, scaling, testing strategy
- Consolidated findings table with severity levels
- Critique report template
- Severity system: 🎯 Must-Address, 💡 Recommendation, 🤔 Question
- Read-only for existing artifacts, creates new critique report

### Requirements

- Spec Kit: >=0.1.0
